Minimal Express + Mongoose API for hotels.

Quick start:

1. Copy `.env.example` to `.env` and set `MONGO_URI`.
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`

API endpoints:
- `GET /api/hotels`
- `GET /api/hotels/:id`
- `POST /api/hotels`
- `PUT /api/hotels/:id`
- `DELETE /api/hotels/:id`
